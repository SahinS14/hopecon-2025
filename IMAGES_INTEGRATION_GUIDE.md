## 🔧 Quick Customization

### Change Images
1. Replace images in `/assets/` subfolder
2. Update `src` paths in HTML files
3. Refresh browser

### Change Logos
```
assets/logos/logo.png          ← Navbar logo
assets/logos/KNFLOGO.png       ← Registration page logo
```

### Change Hero Background
```html
<!-- In index-with-images.html -->
style="background: ..., url('assets/hero_area/banner_hopecon.jpg') ..."
```

### Add More Gallery Images
1. Add images to `assets/gallery/`
2. Add gallery items to the gallery section
